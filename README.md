# Employee Task Manager

A simple Java console-based Employee Task Management System built using core Java concepts.

## Features
- Add Employees
- Add Tasks
- Mark Tasks as Completed
- View Employees and Tasks
- Sort Employees by Salary
- Group Tasks by Employee
- Stream API Operations
- Exception Handling
- Generic Repository

## Tech Used
- Java 21
- OOP
- Collections
- Stream API
- Generics
- Exception Handling
